Google Sheets Configuration

Setup Instructions:

  1. Create a Google Sheet with the following sheets:
    • Products - columns: id, name, category, description, image_url, status, pricing_data, options_data
    • Categories - columns: id, name, description, icon, sort_order
    • Materials - columns: id, product_id, name, price_multiplier, description
    • Finishing - columns: id, name, type, options, prices
  2. Make the sheet publicly viewable (Share > Anyone with the link can view)
  3. Get the Sheet ID from the URL (between /d/ and /edit)
  4. Get an API key from Google Cloud Console (Enable Sheets API)
  5. Enter both values above and click Save
