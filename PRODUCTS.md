## ðŸš€ Customization Guide

### Adding Your Own Products

1. Open `config.js`
2. Find `SAMPLE_PRODUCTS` array
3. Add new product object:

```javascript
{
    id: 'your-product-id',
    name: 'YOUR PRODUCT NAME',
    category: 'banner', // or rigid, adhesive, etc.
    description: 'Product description here',
    image: 'https://your-image-url.com/image.jpg',
    status: 'active',
    materials: [
        { name: 'Material 1', price: 2.50, description: 'Description' }
    ],
    options: {
        printSides: ['single', 'double'],
        // Add your custom options
    },
    pricing: {
        singleSided: { 'Material 1': 2.50 },
        doubleSided: 4.00
    }
}
```
